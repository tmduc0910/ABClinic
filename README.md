- Vào cmd, chạy dòng lệnh sau:
      npm install body-parser json-server
- cd vào thư mục mock-server, chạy lệnh:
      npm run start-auth
- Vào trình duyệt, gõ http://localhost:3000, nếu không báo lỗi là thành công
